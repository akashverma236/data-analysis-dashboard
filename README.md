# Sales Data analysis,

## About the Project
This project is based on sales data analysis using Microsoft Excel.
The goal is to understand sales trends, customer behavior, and product
performance using basic Excel analysis techniques.

https://github.com/akashverma236/vrinda-store-data-analyst-project/blob/main/README.md



## Dataset
 Sample sales data
 Includes order date, product category, sales, quantity,
  customer gender & age group, sales channel, and region/state

## Tools Used
 Microsoft Excel
 Pivot Tables
Pivot Charts
 Basic Excel formulas
 Dashboard

## Work Done
 Month-wise sales analysis
 Product category comparison
 Customer analysis (gender & age group)
 Sales channel performance
 State-wise sales/orders analysis
## Outcome
This project helped me improve my Excel skills in data analysis,
reporting, and visualization and is suitable for entry-level
data analyst or internship roles.
